# Document Contract

- Preserve one-column section order: Summary, Experience, Certifications, Projects, Achievements, Education, Technical Skills.
- Use semantic headings, paragraphs, lists, and hyperlinks in both HTML and Writer output.
- Use the configured theme font, body color, and accent color.
- Set Writer to portrait A4, 210 × 297 mm. Poppler normally reports approximately 595.3 × 841.9 points.
- Keep contact details centered on one line, section hierarchy clear, dates right-aligned, and spacing compact.
- Never hide overflow, clip text, use remote fonts, or alter structure to bypass QA.

HTML is a preview. ODT is the editable document. The tagged PDF is the acceptance artifact.
