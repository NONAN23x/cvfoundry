# Writing Style Rules

Use the selected profile's `Writing-Style.md` only to choose verbs, rhythm, and tone.

- Prefer direct technical language, supported metrics, and compact sentences.
- Avoid hype, decorative adjectives, and generic AI phrasing.
- Never infer facts from style examples.
- Keep ATS clarity above voice.
- If guidance is sparse, write conservatively: clear action, concrete scope, supported outcome.
- For the resume summary, follow the selected profile's writing guidance: write an elevator pitch about professional identity, strongest capabilities, working character, and value—not a recap of project bullets.
